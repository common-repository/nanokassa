Плагин для фискализации чеков согласно 54 ФЗ и облачной онлайн кассы Nanokassa.ru 

== Описание ==

Плагин позволяет вам фискализировать ваши платежи (woocommerce) согласно 54 Федеральному Закону. Перед использованием советуем вам изучить информацию на облачном сервисе аренды онлайн касс Nanokassa.ru

Облачная онлайн касса Nanokassa.ru это:

* Готовая интеграция вашего Wordpress и Woocommerce с нашим сервисом
* Непробиваеая защита и безопасность (уникальное двойное шифрование ваших коммерческих данных AES256-CTR + RSA4096)
* Нерушимая работа (неуязвимые каналы связи, балансировка нагрузки, TIER-III Selectel)
* Отличная стоимость (качественно и практично, подключение к ОФД бесплатно!)
* Хорошая документация (возможность подключить любые сервисы для выполнения 54-ФЗ)

== Installation ==

1. Загрузите папку `nanokassa` в директорию `/wp-content/plugins/` .
2. Активируйте плагин через меню в WordPress.
